Markdown

开头的话：

这个事情其实是很搞的，良久不来更新文章，导致当时就学的糊里糊涂的Markdown语法已经忘的光光。大概还能用出来的也就只剩”**”用来表加粗了。隐约还能记得中文段落开头的缩进要打一串半角还是全角来着的黑科技代码，具体是什么当然已经是浑然不知了。当然，网上此类软文已经不胜枚举，再来重复立个牌子，一是“觉知此事要宫刑”，二是准备加上其他一些与Markdown输入相关的技巧和知识，以后麻麻再也不怕我这什么都记不住的黄鱼脑子了。
