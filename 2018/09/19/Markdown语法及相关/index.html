Markdown语法及相关

2018-09-19
Markdown

开头的话：

这个事情其实是很搞的，良久不来更新文章，导致当时就学的糊里糊涂的Markdown语法已经忘的光光。大概还能用出来的也就只剩”**”用来表加粗了。隐约还能记得中文段落开头的缩进要打一串半角还是全角来着的黑科技代码，具体是什么当然已经是浑然不知了。当然，网上此类软文已经不胜枚举，再来重复立个牌子，一是“觉知此事要宫刑”，二是准备加上其他一些与Markdown输入相关的技巧和知识，以后麻麻再也不怕我这什么都记不住的黄鱼脑子了。

####文字添加超链接

效果见上一段中的知乎原贴示例。具体实现的方法为：使用 **[<u文字</u](link** 的格式。此处省略下>号和下）号。

####引用图片及给图片添加超链接

法一：使用html-img标签。例（引自-2018寄首）：

p align=”center”>img src=”/images/google2018.png” width=”90%”>/p>

此处省略了各个标签的上括号，请自行脑补。注意img的原地址此处填写的是相对路径，当然也可以是绝对路径

法二：Markdown语法 ！[图片文字（可空）]（link），例：

![图片名](http://xxx.xxx.xxx 此处省略了末尾的下括号
